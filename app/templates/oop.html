OBJECT ORIENTED PROGRAMMING USING A CALCULATOR

Encapsulation:

Encapsulation describes the idea of wrapping data and the methods that work on the data within one unit. An example of encapsulation is a class as it encapsulates all the data that is member functions, variables, etc.

class Calculator

""" Calculator class """

def __init__(self, first_num, second_num):
self.first_num = first_num
self.second_num = second_num

def addition():
""" Add """
return self.num_1 + self.num_2
def multiplication():
""" Multiplication """
return self.num_1 * self.num_2

Inheritance:

Inheritance allows inheriting attributes and methods from a parent class. It enables creating subclasses and getting all the functions of our parent class, and allows the ability to overwrite or add completely new functions. without affecting the parent class.

class Calculator

""" Calculator class """

def __init__(self, first_num, second_num):
self.first_num = first_num
self.second_num = second_num

We defined addition and multiplication in the parent class above. Now we are inheriting the calculator class to implement subtraction.

Polymorphism:

It refers to the use of a single type entity (method, operator or object) to represent different types in different scenarios.

class Calculator:

""" Calculator class """

def __init__(self, first_num, second_num):
self.first_num = first_num
self.second_num = second_num

def addition():
""" Add """
return self.num_1 + self.num_2

class Add(Calculator):
""" Add inherited from Calculator"""
def addition (self):
""" Add from child class"""
return self.first_num + self.second_num

Abstraction:

Abstraction in OOP is a process of hiding the real implementation of the method by only showing a method signature.

class Calculation:

""" Abstract base class for Calculations"""

# pylint: disable=too-few-public-methods
def __init__(self, values):
self.values = values

@classmethod def create(cls, *tuple_args: tuple):

""" create factory class method """

class Add(Calculator):

""" Add inherited from Calculator"""

# pylint: disable=too-few-public-methods
@classmethod def create(cls, *tuple_args: tuple):

""" create factory class method """
values = list(tuple_args)
return cls(values)

def add(self):

""" Add from child class"""
result = 0
for val in self.values:
result = val + result
return result
